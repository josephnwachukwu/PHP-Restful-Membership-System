Restful Membership System

A Restful Services System for membership using php

View project on GitHub

Speed Up your Development

Speed up your development with this easy to use and easy to install restful backend system written in php and mysql

Features:

  • Full Authentication - The script provides the complete functionality for authentication: Log in, Sign up, Email Activation and Password Recover.
  • User Settings - The users have access to a settings page where they can change the email address, username, password, profile information
  • Advanced Mailer - Use a SMTP configuration, navite PHP mail
  • Restful Endpoints - All of the form are submitted through REST to minimize page refreshes and for a better user experience.
  • Admin Contrl Panel - Manage users, roles, custom fields, script settings, send emails and private messages.
  • User Roles & Permissions - Create multiple roles with permissions and assign them to the users.
  • Security - The script uses Bcrypt for password hashing, encrypted cookies, XSS and CSRF attack preventions.
